Está no terminal de texto e precisa alterar entre os terminais de texto:

<alt> + F1 - /dev/tty1 ou
<alt> + F2 - /dev/tty2 ou
<alt> + F3 - /dev/tty3 ou
<alt> + F4 - /dev/tty4 ou
<alt> + F5 - /dev/tty5 ou
<alt> + F6 - /dev/tty6

O GNU/Linux possui 64 terminais onde seis são controlados pelo usuario e outros usados para visualização.
Para acessar tty9, tty10, tty11 e tty12 usa-se as teclas de função F9 a F12


Limpar a tecla
#clear

ou pelo atalho
<ctrl> + <l>


Comandos para logoff
#exit

ou

#logout

ou pelo atalho

<ctrl> + <d>


Comandos para Reiniciar o sistema GNU/Linux

#reboot

ou

#systemctl reboot

ou

#init 6

ou

#echo b > /proc/sysrq-trigger

ou

#shutdown -r now



Comandos para Desligar o sistema GNU/Linux

#halt 	#Não é o mais recomendado
	#Instrui o hardware para parar 		todas as funções deixando a CPU 	ligada

#poweroff 	#Instrução mais adequada 			para o desligamento
	

#shutdown -h now


Comando para desligar e/ou reiniciar

#shutdown -r now
#shutdown -h now

ou

#shutdown -r +5	#Reinicia daqui 5 minutos
#shutdown -h +5	#Desliga daqui 5 minutos

ou

#shutdown -r 13:30	#Reinicia na hora programada
#shutdown -h 13:30	#Desliga na hora programada

ou

Reinicia daqui 5 minutos com mensagem aos usuarios logados
#shutdown -r +5 "Mensagem"		

Desliga daqui 5 minutos com mensagem aos usuarios logados
#shutdown -h +5	#Desliga daqui 5 minutos
